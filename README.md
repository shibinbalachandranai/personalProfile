# Shibin Balachandran — Personal Portfolio Website

A modern, static personal portfolio website for Shibin Balachandran — Technology Leader, AI Enthusiast, Mentor & Business Consultant.

## Overview

This is a single-file static HTML/CSS/JS portfolio built with a dark, modern design using gradient accents and smooth animations. No frameworks or build tools required — just open in a browser.

## File Structure

